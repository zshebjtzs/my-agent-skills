# 参考文档模板示例

> 本文档为 AI Agent 提供技能所需的外部参考信息，例如项目依赖、API 接口说明、数据格式规范等。
> Agent 在开发或执行本技能时，如需查询外部信息或判断是否可复用现有资源，应参考本文档。

## 📌 使用说明
本目录 (`references/`) 用于存放与技能相关的补充文档。你可以在此添加：
- 技术依赖列表（如 npm、pip 包及用途）
- 外部 API 的认证方式、端点列表
- 数据库表结构或字段说明
- 操作规范或配置模板

以下是一个示例 —— 假设本技能需要操作一个前后端项目，可以列出关键依赖包以便 Agent 了解可用的工具库。
