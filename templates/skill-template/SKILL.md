# 技能模板 — 使用指南

> 本模板用于快速创建新的 Agent Skill。所有示例文件（`scripts/`、`references/`、`assets/` 下的内容）均为占位说明，**实际使用时应删除或替换**。

---

## 📂 模板目录结构

```text
skill-template/                # 复制后重命名为你的技能名
├── SKILL.md                   # 本文件（技能定义 + 说明）
├── scripts/                   # 存放可执行脚本（Python/Shell等）
│   └── example-script.py      # 示例脚本（可删除）
├── references/                # 存放参考文档、API说明等
│   └── example-reference.md   # 示例文档（可删除）
└── assets/                    # 存放静态资源（图片、模板等）
    └── example-asset.txt      # 示例资源（可删除）
```

## 🚀 创建新技能的步骤

1. **复制本模板**：将 `templates/skill-template/` 整个目录复制到 `skills/` 下的目标分类（如 `shared/`、`personal/`），并将目录重命名为你的技能名称（例如 `pdf-analyzer`）。
2. **编辑 `SKILL.md`**：
   - 修改 YAML 元数据（`name`、`description`、`author` 等）。
   - 重写正文部分，给出具体、可执行的指令（参考下文【编写指南】）。
3. **按需添加或删除子目录**：
   - 如果需要脚本、参考文档或资源，请保留对应目录并放入实际文件。
   - 如果不需要，可以删除整个子目录。
   - **务必删除所有 `example-*` 示例文件**，避免干扰。


## 📝 技能编写指南（给 Agent 的指令）

在 `SKILL.md` 的正文中，请清晰描述：

- **技能目标**：解决什么问题？
- **输入要求**：用户需要提供什么？
- **执行步骤**：分步说明（如调用哪个脚本、读取哪个文件、输出什么）。
- **输出格式**：最终返回的结果类型（纯文本、JSON、表格等）。

### 示例正文片段

## 技能目标
分析 PDF 文件，提取表格数据并生成 CSV。

## 输入要求
用户提供 PDF 文件路径。

## 执行步骤
1. 运行 `scripts/extract.py` 读取 PDF。
2. 将数据转换为 DataFrame。
3. 输出 CSV 到当前目录。

## 输出格式
返回 CSV 文件路径。


## 🧹 清理提醒

完成技能开发后，请确认已删除所有示例文件。如果某个子目录为空，也可以直接删掉该目录，保持仓库整洁。

**现在，请开始创建你的专属技能吧！**
